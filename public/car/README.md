# Car images for location pages

Place 2–3 or more car images here so each branch can display them in the location detail slider.

**Required files (used by default):**

- `car1.jpg`
- `car2.jpg`
- `car3.jpg`
- `car4.jpg`
- `car5.jpg`

Add more files (e.g. `car6.jpg`, `car7.jpg`) and register them in `src/lib/locationImages.ts` in the `CAR_IMAGES` array if you want a larger pool.

**Recommended:** 1200×800 or similar aspect ratio (16:10) for best display in the slider.
